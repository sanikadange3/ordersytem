OrderEase – Online Food Ordering Demo
ordersystem-a8e41.web.app

OrderEase is a simple online food ordering system designed for demonstration purposes. Users can view a menu, add items to a cart, and place orders. The system includes login functionality and a cart system, but does not use any database — all data is stored locally or in-memory.

Features
User Features
Login: Access with predefined credentials (username: user1, password: pass123).
View Menu: Browse available food items with name, price, description, and image.
Add to Cart: Add items to your cart and modify quantities.
Place Order: Confirm orders and see a success message.
Admin / Demo Features
Manage Menu: Add, edit, or remove menu items for the current session (in-memory only).
Real-Time Updates: Changes reflect instantly in the menu and cart.
Technical Details
Frontend: HTML, CSS, JavaScript (optional: React for dynamic UI)
Backend: Optional Node.js or Python Flask (all data handled in-memory)
Storage: No database used; all data is stored locally or in-memory.
How to Run
Clone or download the repository.
Open index.html in a web browser.
Login using the credentials:
Username: user1
Password: pass123
Browse the menu, add items to the cart, and place your order.
Notes
Data is not persistent: Orders and menu changes will be lost on page refresh.
For demonstration only: Ideal for testing UI and flow without a database.
Future Enhancements
Integrate a real database (Firebase, MySQL, etc.) for persistent storage.
Add payment functionality.
Include order notifications.
